# MUBI-IPC-Actividad-grupal

## Descripción

Actividad grupal de Introducción a la Programación Científica.

## Propósito

Este proyecto tiene como objetivo crear un entorno de trabajo compartido en GitHub y gestionar el control de versiones.

## Objetivos

* Crear un repositorio en GitHub.
* Generar forks para realizar modificaciones del main y solicitar pull request.
* Gestionar el control de versiones y admitir o rechazar cambios.
* Consensuar la estructura en carpetas y archivos del repositorio final.

## Estructura del proyecto

* `datos/`: contiene los datos de los alumnos
* `reports/`: contiene los markdown / html de presentación de resultados
* `scripts/`: contiene los scripts
* `Imagenes/`: contiene los gráficos

## Herramientas

* Git y GitHub
* R
* Software de análisis bioinformático
