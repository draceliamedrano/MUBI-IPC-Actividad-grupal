## MUBI-IPC-Actividad-grupal

# Crear un repositorio github. Control de versiones
Actividad grupal para crear un entorno de trabajo compartido

# Estructura del sitio; árbol de carpetas: 
Contaremos con las siguientes: 
  
- datos: contendrá los archivos de datos en bruto
- reports: contendrá los markdown / html de presentación de resultados
- scripts: contendrá los archivos de código según se vayan agregando o modificando, y se acepten
- .gitignore: elementos que no deseamos que se incorporen
- LICENSE: tipo de licencia que se aplique
- README.md: donde nos encontramos ahora; explicación del objeto del trabajo
